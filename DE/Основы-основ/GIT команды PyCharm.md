таблица соответствий между **Git CLI** и **PyCharm**, чтобы сразу видеть, что делает каждая операция визуально.

|Действие|Git CLI|PyCharm GUI|Комментарий|
|---|---|---|---|
|Подготовка файла к коммиту|`git add <file>`|Отметить файл в **Commit Window**|Аналог staging area. Выбираете, что попадёт в коммит.|
|Подготовка части изменений в файле|`git add -p <file>`|Выделить фрагменты в **Diff/Partial commit**|Можно коммитить отдельные изменения в файле.|
|Создание коммита|`git commit -m "message"`|Нажать **Commit** в Commit Window|PyCharm создаёт коммит с выбранными файлами.|
|Создание коммита и отправка на remote|`git commit -m "message" && git push`|**Commit and Push**|Одновременно создаёт коммит и пушит в remote.|
|Просмотр изменений|`git status`, `git diff`|Панель **Version Control → Local Changes**|Видно изменённые файлы и конкретные изменения.|
|Отмена изменений|`git checkout -- <file>`|**Rollback** через контекстное меню на файле|Возвращает файл к последнему коммиту.|
|Изменение последнего коммита|`git commit --amend`|**Amend Commit** в Commit Window|Добавить забытые изменения в последний коммит.|
