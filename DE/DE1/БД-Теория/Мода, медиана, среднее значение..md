**Меры центральной тенденции** — это статистические показатели, которые описывают «центр» или типичное значение набора данных. Они отвечают на вопрос: «Вокруг какого значения группируются мои данные?». Выбор правильной меры критически важен для корректного анализа.


| Мера                              | Определение                                                                                                                                       | Когда использовать?                                                                                                                    | Пример на: [5, 1, 3, 7, 3, 100]                                                                                                       |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------- |
| **Среднее арифметическое (Mean)** | Сумма всех значений, деленная на их количество.  <br>`mean = sum(x_i) / N`                                                                        | Когда **распределение данных симметричное** (например, нормальное) и **нет выбросов**. Чувствительно к экстремальным значениям.        | `(5+1+3+7+3+100) / 6 = 119 / 6 ≈ **19.83**`                                                                                           |
| **Медиана (Median)**              | Значение, которое делит упорядоченный по возрастанию набор данных **пополам**. 50% данных лежит ниже медианы, 50% — выше.                         | Когда распределение **скошенное (асимметричное)** или есть **выбросы**. Устойчива (робастна) к экстремальным значениям.                | Упорядочиваем: `[1, 3, **3**, **5**, 7, 100]`.  <br>Четное число элементов: медиана = среднее между 3 и 5.  <br>`(3 + 5) / 2 = **4**` |
| **Мода (Mode)**                   | Наиболее **часто встречающееся** значение в наборе данных. Набор может быть ** unimodal** (одна мода), **bimodal** (две моды) или **multimodal**. | Для **категориальных (номинальных) данных** (например, самый популярный цвет товара). Также для пиковых значений в непрерывных данных. | Частоты: `1->1, 3->2, 5->1, 7->1, 100->1`.  <br>Наиболее частое значение: **3**                                                       |
