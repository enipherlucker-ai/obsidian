# Функции первого класса и функции высшего порядка

## Функции первого класса
Функции в Python (и многих других языках) являются **функциями первого класса**.  
Это значит, что их можно:
- присваивать переменным,
- передавать в другие функции как аргументы,
- возвращать из функций как значения,
- хранить в структурах данных (списки, словари и т. д.).

Пример:
```python
def greet(name):
    return f"Привет, {name}!"

hello = greet  # присвоение функции переменной
print(hello("Аня"))
```

# Функции порядка. Что это такое и какие бывают

## Функции высшего порядка

**Функции высшего порядка** — это функции, которые:
- принимают другие функции в качестве аргументов,  
- возвращают функции как результат,  
- либо делают и то, и другое.  

Чаще всего такие функции реализуют обобщённый алгоритм (например, сортировку), а ключевую часть логики делегируют программисту.  
Главный плюс — **сокращение дублирования кода**.

---

## Примеры


### 1. `reduce` — свёртка списка

```python
from functools import reduce  

items = [1, 2, 3, 4, 5]  
print(reduce(lambda x, y: x * y, items))  
# Результат: 120
```

### 2. `map` + `filter` + `sum`

```python
items = [1, 2, 3, 4, 5, 6]  
print(sum(map(lambda x: x * x, filter(lambda x: x % 2 == 0, items))))  
# Результат: 56 (2² + 4² + 6²)

```

### 3. Поиск самого длинного слова

```python
items = ["кот", "собака", "носорог", "ёж"]  

def entice(str1, str2):  
    if len(str1) > len(str2):  
        return str1  
    return str2  

print(reduce(lambda x, y: entice(x, y), items))  
# Результат: "носорог"
```

### 4. Группировка слов по первой букве

```python
items = ["кот", "кит", "собака", "слон", "ёж"]  
dict = {}  

def add_item_in_dict(tmp_dict: dict, item: str):  
    if item[0] in tmp_dict:  
        tmp_dict[f"{item[0]}"].append(item)  
    else:  
        tmp_dict[f"{item[0]}"] = [item]  
    return tmp_dict  

print(reduce(lambda x, y: add_item_in_dict(x, y), items, dict))  
# Результат: {'к': ['кот', 'кит'], 'с': ['собака', 'слон'], 'ё': ['ёж']}

```

### 5. `map` для возведения в квадрат

```python
items = [1, 2, 3, 4, 5]  

items_sqr = list(map(lambda x: x * x, items))  
print(f"old is {items}")  
print(f"new is {items_sqr}")  
# old is [1, 2, 3, 4, 5]  
# new is [1, 4, 9, 16, 25]

```

### 6. `filter` для отбора чётных чисел

```python
items = [1, 2, 3, 4, 5, 6, 7, 8]  

items_filt = list(filter(lambda x: x % 2 == 0, items))  
print(f"old is {items}")  
print(f"new is {items_filt}")  
# old is [1, 2, 3, 4, 5, 6, 7, 8]  
# new is [2, 4, 6, 8]

```

### 7. Подсчёт суммарной длины слов

```python
items = ["кот", "собака", "ёж"]  
print(sum(list(map(lambda x: len(x), items))))  
# Результат: 10 (3 + 6 + 1)

```

### 8. Передача функции в функцию (`twice`)

```python
def twice(func, x):  
    return func(func(x))  

print(twice(lambda x: x + 2, 5))  
# Результат: 9
```
